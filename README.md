# EEG-Based-Depression-Detection

This project detects depression using EEG signals and machine learning techniques.

## Dataset
This project uses a publicly available EEG dataset.
Source:
https://figshare.com/articles/dataset/EEG_Data_New/4244171/2

The dataset contains EEG recordings in EDF format for:
- Healthy (H)
- Major Depressive Disorder (MDD)

After downloading, organize the dataset.
